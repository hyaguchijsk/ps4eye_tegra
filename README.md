# ps4eye_tegra

## Description

PS4Eye proc for Tegra K1.
This is developped and tested on Jetson TK1.

## How to Use

```
$ roslaunch ps4eye_tegra stereo.launch
```

to open camera and process stereo matching, create point cloud.
